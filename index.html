Get Started

Twiff Edu.

Website kami bernama twiff.edu, website ini merupakan website media pembelajaran programming yang berisi berbagai macam tutorial programming seperti HTML,CSS,Javascript, dan Phyton. Terdapat tutorial dalam bentuk video dan ada juga yang berbentuk teks.

Background

Di era digital dan perkembangan teknologi yang pesat seperti sekarang ini profesi programmer atau developer perangkat lunak banyak menarik minat orang. Saat ini sudah banyak orang yang mempelajari coding, walaupun bukan dari background yang memiliki pendidikan komputer atau informatika. Namun, terkadang saat mempelajari suatu codingan orang-orang terkendala karena Minimnya website pembelajaran Programming yang menggunakan Bahasa Indonesia. Selain itu juga banyak orang-orang yang terkendala dengan kurangnya informasi yang didapatkan terkait media pembelajaran programming. Ada juga website yang menyediakan pembelajaran programming namun menggunakan bahasa yang sukar untuk dipahami.

Purpose

Untuk memberikan suatu media pembelajaran programming yang mudah dipahami dan menggunakan bahasa yang umum digunakan. Website kami hadir agar orang-orang yang ingin belajar programming mendapatkan media dengan berbagai tutorial codingan.

Services

Here is the tutorial we serve

HTML5

Hypertext Markup Language atau HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman website. HTML terdiri dari kombinasi teks dan simbol yang disimpan dalam sebuah file. Untuk bisa diakses secara umum, pengguna perlu membukanya lewat aplikasi browser, seperti Internet Explorer, Chrome, atau Mozilla Firefox.

CSS3

CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.

JavaScript

JavaScript adalah bahasa pemrograman populer yang digunakan untuk membuat situs dengan konten website yang dinamis. Konten dinamis artinya konten dapat bergerak atau berubah di depan layar tanpa perlu mereload halaman.

Python

Python adalah salah satu bahasa pemrograman yang dapat melakukan eksekusi sejumlah instruksi multi guna secara langsung (interpretatif) dengan metode orientasi objek (Object Oriented Programming) serta menggunakan semantik dinamis untuk memberikan tingkat keterbacaan syntax.
