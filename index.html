Twiff.Edu

Website kami bernama Twiff.edu, website ini merupakan website media pembelajaran programming yang berisi berbagai macam tutorial codingan seperti HTML,CSS,Javascript, dan Phyton. Terdapat tutorial dalam bentuk video dan ada juga yang berbentuk teks.

Background

Saat mempelajari suatu bahasa pemrograman orang-orang terkendala dengan Minimnya website pembelajaran Programming yang menggunakan Bahasa Indonesia. Selain itu juga orang-orang terkendala dengan kurangnya informasi yang didapatkan terkait media pembelajaran programming. Ada juga website yang menyediakan pembelajaran programming namun menggunakan bahasa yang sukar untuk dipahami.

Purpose

Untuk memberikan suatu media pembelajaran programming yang mudah dipahami dan menggunakan bahasa yang umum digunakan. Website kami hadir agar orang-orang yang ingin belajar programming mendapatkan media dengan berbagai tutorial codingan.

Services

Tutorial yang kami sediakan

HTML5

Hypertext Markup Language atau HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman website. HTML terdiri dari kombinasi teks dan simbol yang disimpan dalam sebuah file.

CSS3

CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.

JavaScript

JavaScript adalah bahasa pemrograman populer yang digunakan untuk membuat situs dengan konten website yang dinamis. Konten dinamis artinya konten dapat bergerak atau berubah di depan layar tanpa perlu mereload halaman.

Python

Python adalah salah satu bahasa pemrograman yang dapat melakukan eksekusi sejumlah instruksi multi guna secara langsung (interpretatif) dengan metode orientasi objek serta menggunakan semantik dinamis untuk memberikan tingkat keterbacaan syntax.
